This solidity file is about simple Banking-Depositing, viewing you balance, and withdrawing.
1. Go to www.remix.ethereum.org
2. Copy and Paste the StateMachine_SelfDestruct.sol file into the web (www.remix.ethereum.org)
3. Under the Compile Tab at the upper right hand side of the remix, Select the compiler version to 0.4.24
4. Click Start to Compile or CTRL + S
5. Go to the Run Tab beside Compile Tab and click Deploy
6. Once the Deploy is clicked, you would see that there's a Deployed Contracts, click the arrow beside the deployed contract (left side)
7. You would see a clickable deposit, destroyContract, withdraw, and viewBalance
8. If you want to deposit, set the Value and beside that click on wei and change it to Ether.
9. To see your balance, just click on viewBalance.
10. To withdraw your remaining balances, set the withdraw to wei unit and click withdraw.
11. To terminate this contract, just click on destroyContract.
